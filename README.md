haha
====
数据监控面板（像ducksboard）

![Orient](https://github.com/tigerlove/haha/doc/demo.png)
## Requirements

* Ruby2.1.0
* Rails 4.0+
* Mongodb

## 数据格式

现在支持pie，line，area，column，bar图jsonp数据,除pie图外，可以有多列

```
[["浏览器", "浏览器占比"], ["firefox", "578.0"], ["chrome", "278.0"], ["IE8", "477.0"], ["IE7", "177.0"], ["360", "777.0"]]
```
